## ADDED Requirements

### Requirement: Enhanced post header (Hero Banner)
文章頂部應顯示一個色彩豐富的 Hero Banner，背景色隨分類變化。

#### Scenario: Visual separation from content
- **WHEN** 進入文章頁面
- **THEN** 系統顯示具備漸層背景的標題區域，並與下方正文有視覺分隔線

### Requirement: Code block macOS decoration
程式碼區塊頂部應顯示模擬 MacOS 視窗的三色按鈕。

#### Scenario: Code aesthetics
- **WHEN** 文章中包含程式碼區塊
- **THEN** 區塊頂部出現灰色飾條與紅、黃、綠三個圓點

### Requirement: Progressive scroll tracking
頁面應提供捲動進度指示，讓使用者知道目前閱讀位置。

#### Scenario: Scroll percentage display
- **WHEN** 使用者向下捲動頁面
- **THEN** 系統在角落或頂部顯示即時的閱讀百分比
