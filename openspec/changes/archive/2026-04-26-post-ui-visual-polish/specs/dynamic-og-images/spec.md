## ADDED Requirements

### Requirement: Per-post dynamic OG image
系統應為每篇內容產生包含標題、標籤與網站品牌的專屬分享圖 (OG Image)。

#### Scenario: Metadata inclusion in OG image
- **WHEN** 系統產生文章的 OG 圖
- **THEN** 圖片中必須包含該文章的正式標題與主要分類標籤

### Requirement: Consistent visual style for social sharing
產出的圖片應符合全站的技術美學風格。

#### Scenario: Visual consistency
- **WHEN** 圖片被分享至社群媒體
- **THEN** 其配色、字體應與網站本體保持一致
